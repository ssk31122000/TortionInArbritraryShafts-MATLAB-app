User Guide of App

The app deals with the torsion of arbitrary shafts, in which user needs to give the equation of cross section as input and the app will show the useful plots and results. It has two windows: Standard Shapes window and Arbitrary Shapes window.

Standard Shapes window:

Input in Standard shape window: Radius of circle or major-minor axis of ellipse from shape selection panel, material selection panel and angle of twist per unit angle.

Output in Standard shape window: Displacement fields (u,v,w), shear stresses (σzx,σzy), warping funtion plot, stress distribution plot, Torque (in N-m).

Steps need to be followed:

Step 1: Choose one shape out of circle and ellipse and input the radius of circle (in meter) or major and minor axis of circle (in meter). The range for radius is from 0.5m to 5m and range for both axis of ellipse are from 1 to 10 meter.

Step 2: Choose one material from the material panel, the value of shear modulus (G) will set after selection.

Step 3: Input the value of angle of twist per unit length (θ) and click on the calculate button.

All the required outputs will display at that time, and plots will display within 1 minute.

REQUIRED OUTPUT (Displacement and Stress fields)

REQUIRED PLOTS IN THE RANGE OF X&Y FROM [-5,5], ORIGIN(0,0)

Step 4: In case of no output/plots, see the 'Check panel' on the window to know the reason for not showing any output or plot. It will check the input limits, satisy compatibility equations or not, etc.

Refresh Button: Click the refresh button to clear the screen.

Arbitrary Shapes window:

Input in arbitrary shape window: Cross sectional equation in the form of F(x,y)=0, material selection panel and angle of twist per unit angle.

Output in Standard shape window: Displacement fields (u,v,w), shear stresses (σzx,σzy), 2D cross section plot, warping funtion plot, stress distribution plot.

Steps need to be followed:

Step 1: Input the cross sectional equation in the form of F(x,y) =0. For example, if cross sectional equation is ax2 + by2 = cx + dy +e, then user need to input ax2 + by2 - cx - dy - e as input function.

Step 2: Choose one material from the material panel, the value of shear modulus (G) will set after selection.

Step 3: Input the value of angle of twist per unit length (θ) and click on the calculate button.

All the required outputs will display at that time, and plots will display within 1 minute.

REQUIRED OUTPUT (Displacement and Stress fields)

REQUIRED PLOTS IN THE RANGE OF X&Y FROM [-5,5], ORIGIN(0,0)

Step 4: In case of no output/plots, see the 'Check panel' on the window to know the reason for not showing any output or plot. It will check the input limits, satisy compatibility equations or not, etc.

Refresh Button: Click the refresh button to clear the screen.

Few Limitations of the app:

  1. MATLAB should be pre-installed to use the app.
  2. Require Internet browser to view theory manual, data dictionary and user manual.
  3. Require internet connectivity to view few images in the manual which are directly taken from the web.
  4. It takes around 1 minute to visualise the plot because each plot computes value over 2500 points.
  5. Standard Shapes window is limited to circle and ellipse. Torsion problems based on cross sections of other shapes need to be solved through arbitrary shape window which require cross sectional equation.
  6. Code is not computing twisting moment for prismatic bars of arbitrary cross section. It is computing twisting moment only for circle and ellipse in standard shapes window.
  7. Radius of circle in standard shapes window is limited to the values between 0.5 and 5 meters.
  8. Major axis and minor axis of ellipse in standard shapes window is limited to the values between 1 and 10 meters.
  9. Mesh grid is chosen from -5 meter to 5 meter, thus only user can visualise the plot between that range only. [-5m < x < 5m, -5m < y < 5m]
