***INSTALLATION GUIDE***

TO install and use this app, user will require MATLAB R2020b

To install Tortion in Arbitrary Shafts MATLAB app, double click on the MLAPPINSTALL file
this will automatically install the app in your matlab app library


If this does not work then,

open MATLAB->go to apps->Install app

To open installed app

open MATLAB->go to apps->open Tortion in Arbitrary Shafts 



***TO OPEN SOURCE CODE***

TO view full source code, user will require MATLAB R2020b

open MATLAB->go to apps->go to Design apps->click open in App Designer window->select 'FinalApp_1' in the source code folder
